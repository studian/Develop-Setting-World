# OpenCV 3.2 Viz Setting

## Env
* OS : windows 10 64 bit
* VS : Visual Studio 2017 
* Cmake : 3.8.2
* VTK : 8.0.1
* Eigen : 3.3.4 (stable version)
* OpenCV : 3.2
* OpenCV_Contrib : 3.2
